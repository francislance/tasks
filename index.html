Welcome to the Tasks Helm Chart Repository

This repository contains the Helm chart for deploying both frontend (ReactJS) and backend (Kotlin) applications of the tasks project.

Using the Helm Chart

To add this Helm repository to your local Helm client, use the following command: 

helm repo add tasks https://francislance.github.io/tasks

Once added, you can install the chart by running:

helm install my-tasks tasks/tasks

Available Values

You can override the default values in the chart by providing your own values.yaml file. Some common overrides include:

  • frontend.replicas: Number of frontend replicas.
  • backend.replicas: Number of backend replicas.
  • frontend.image.repository: Docker image for frontend.
  • backend.image.repository: Docker image for backend.
